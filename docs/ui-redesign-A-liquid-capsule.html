One shape, one motion, one curve.

Every mode change is rendered as a single liquid morph of the same vessel — width, height, and corner radius interpolate together on one master easing (cubic-bezier(0.32, 0.72, 0, 1), Apple's nav curve). Content cross-fades on the exact same duration so the eye reads "one motion", not "shape changing while stuff appears". First-open jank is killed by promoting the bubble to its own GPU layer up-front (will-change: transform, border-radius) and by using no backdrop-filter on the large surface — only the compact island has glass.
