Direction C · Spatial Tabs

Zones, not modes. Scale, not morph.

The bubble is one container with stacked zones — a status header on top, a body underneath. When mode changes, the active zone simply scales up by 4% while content cross-fades on the same curve. No shape morph, no border-radius interpolation, no fighting transitions. Total motion budget per mode-switch is ~360ms of one composited animation. Settings ditches expanders entirely for a command-palette-style flat list — searchable, every setting one click away. Power users live here.
