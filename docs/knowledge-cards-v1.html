v1 ↔ v0 主要变化
  • 生成不再占用 bubble 主状态 — generating_cards 状态删除。controls 里的 📚 按钮自身 pulse 即正在生成，主流不阻塞。
  • dashboard 模式整个砍掉。pending queue 在有请求时自动展开为 approval；sessions / 审计 / pairing 全部纳入 settings 的可折叠 section。
  • Knowledge Cards 单开为 cards 模式（460 × 600），三 tab：Today / History / Wrong book。多邻国式的每日目标 + 连击数。
  • 每天的 abstract 用 markdown theme 渲染；可一键导出 .md（today / 全量历史 / 错题本三种）。
  • 所有 Duolingo 借鉴项均用户可关：daily goal 滑块（1-15）、auto-add 错题本、auto-generate、连击通知。
  • Settings · Cards 顶部新增 ✦ Learning focus 多行输入 —— 用户写一段"我想学什么"，prompt 生成时拼到 claude -p 前面，让 abstract 和卡片都带这个 bias。空着 = 让 AI 自己挑。abstract 顶部有 focus 小胶囊提示当时用了什么 focus；导出 markdown 顶部 yaml frontmatter 也写一行。
  • 难度三档：easy 概念性 / medium 基础理论 + 实现方式 / hard 数学基础 + 底层原理。每张卡贴 chip；Settings 里 segmented control 选生成时的偏好（Casual / Balanced / Deep）；Today tab 进度条旁显示当日难度分布。
  • 强制溯源（hard-coded）：source 区从静态一行升级为可展开 quote block，里面是 session 当时的真实对话片段（user / assistant / edit 三种说话方角色色区分）。Wrong feedback 的 "AI explanation" 删掉，改为 From session: <verbatim quote>。实现层硬约束每张卡 JSON 必须有 source.snippet verbatim 字段，没有就丢弃；不向用户暴露关闭开关 —— 防止误关后看到假题。
  • 空白日 fallback：今天没 session 时 Today tab 自动切到 "No new sessions" 变体，从过去 N 天（用户可设 3/7/14/30）+ 错题本里抽题，CTA 文案变 "Start replay"。Streak 显示 🛡 protected 不算断（多邻国 streak freeze 借鉴）。题源仍 100% 来自历史真实 session。

为什么这样改
  • 旧 dashboard 把 sessions / pending / 审计 / pairing 全部展开，420 × 540 屏被填得密密麻麻。Pending 反正会自动展开 approval 卡，没必要再列一遍。

10 / 导出对话框 点击任意 ↓ md 按钮弹出预览模态。代码块用 mono 字体高亮 token（accent / warm / sage 区分 markdown 句法）。底部显示推断的文件名 + Copy / Save 二选一。直接落盘 ~/Downloads/ 或者复制到剪贴板贴进 Obsidian / Notion。
导出格式约定
  • 日期作为 H1 (#)，session 概要为 H2 (##)，子分类 H3。
  • 每张卡用 **Q** + **A** 双行格式，方便后续导入 Anki 时简单拆分。
  • All-history 导出用 --- 横线分隔每天，方便 Obsidian 单文件 outline。
  • 导出前再过一次脱敏：去掉 .env 路径段落、token-like 字符串、绝对路径里的用户名。
  • 默认文件名 companion-YYYY-MM-DD.md / companion-history.md / companion-wrong-book.md。
