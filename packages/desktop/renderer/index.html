Knowledge Cards 数据使用说明

Companion 会读取 daemon 保存的 过去 N 天 Claude Code session 内容（user prompts、assistant 回答、tool 调用 + 输出摘要），过一遍脱敏：

  • 去掉 .env / .envrc / secrets/ 路径附近的行
  • 替换 token-shaped 字符串（GitHub PAT / Anthropic key / AWS key 等）
  • 用户名 → ~

然后通过 stdin 喂给本机的 claude -p 子进程生成卡片。

数据流

  • 不直接调 Anthropic API（用你已认证的本地 Claude Code）
  • 不持久化 prompt（生成的卡片片段会作为 source 留下）
  • 不联网 —— 除非你开了 Allow web fallback

预计 token 用量

3-15k input tokens 每次（取决于活动量、是否启用 web fallback）

同意后默认沿用此设定。可在 Settings 里"Reset consent"重新弹出。
